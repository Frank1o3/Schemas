# SmithPy Modrinth API Config Schema

Schema for Modrinth API endpoint configuration

## Configuration Flags

### `BASE_URL`

**Type:** `string`

Base Modrinth API URL

**Constraints:**
- Pattern: `^https://`

### `ENDPOINTS`

**Type:** `object`

Modrinth API endpoint mappings
