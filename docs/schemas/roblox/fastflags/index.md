# Roblox Fast Flags

Validation schema for Roblox ClientAppSettings Fast Flags
### `FFlagDebugGraphicsPreferVulkan`
- Type: `boolean`
- Description: Prefer Vulkan rendering backend
- Allow-list status: `allowed`

### `DFIntTextureQualityOverride`
- Type: `integer`
- Description: Overrides texture quality level
- Allow-list status: `allowed`

### `DFIntTaskSchedulerTargetFps`
- Type: `integer`
- Description: Community FPS scheduler flag
- Allow-list status: `not-allowlisted`
- Notes: Ignored by current Roblox clients
